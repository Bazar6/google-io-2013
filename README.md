# Deps

## Guard (file watcher) & Compass

```
gem install bundler
bundle install
```

## gjslint

```
sudo easy_install http://closure-linter.googlecode.com/files/closure_linter-latest.tar.gz
```

## jscoverage

```
brew install jscoverage
```

# Running

Start guard: `bundle exec guard`
